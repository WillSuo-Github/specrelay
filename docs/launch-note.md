# SpecRelay v0.1.0

SpecRelay is a small static tool for founders using AI app builders such as Lovable, Bolt, Replit, Cursor, or v0.

The first MVP turns an AI-built prototype description into:

- launch readiness score
- priority risk list
- QA checklist
- developer handoff brief
- downloadable Markdown packet
- review questions for first evaluator feedback

Live app: https://willsuo-github.github.io/specrelay/

The goal of this release is not to claim automated code scanning. It tests a narrower question: whether founders and contractors want a structured handoff packet before a customer pilot, contractor scoping call, or launch review.

Looking for 5 founders, contractors, or agencies who have reviewed an AI-built prototype and can tell me whether this packet would save a handoff call.
