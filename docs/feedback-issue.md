# Feedback Wanted: Does this handoff packet save a scoping call?

SpecRelay v0.1.0 is live: https://willsuo-github.github.io/specrelay/

I am looking for 5 founders, contractors, or agencies who have built or reviewed an AI-built app prototype.

The question: would a structured handoff packet like this save time before a contractor scoping call, customer pilot, or launch review?

## Useful Feedback

If you try it, please share:

1. Did you use a sample packet or your own app details?
2. Which output was most useful: readiness score, risk list, QA checklist, engineering brief, or review questions?
3. What information would still be missing before an engineer/agency could scope the next pass?
4. Would you use this before sending a Lovable/Bolt/Replit/Cursor/v0 prototype to someone else?
5. What would make this worth paying for: better export, automated URL scan, contractor intake, or expert review?

## Current Scope

This first release is intentionally manual and static. It does not scan code or claim security coverage. It is testing whether the handoff packet itself is valuable.
